    Summary : Latitude vs X

    The purpose of this project was to analyze how weather changes relative to distance from the equator. To accomplish this analysis, weather data was queried via the OpenWeather API, for 500 cities chosen at random for study.

    After assembling the data set, Matplotlib was used to the plot the various aspect of the weather vs. latitude. Factors considered were temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
